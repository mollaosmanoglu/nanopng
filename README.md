# nanopng

Generate transparent PNG images from text prompts using Nano Banana Pro + background removal.

## Installation

```bash
uv pip install -e .
```

## Usage

```bash
nanopng "hydra mythological creature"
```

## Tech Stack

- Nano Banana Pro 3.1 (Gemini 3 Pro image generation)
- remove.bg API (background removal)
- Python 3.12+
